delete/v1/disk/resources
Удалить файл или папку

Заметки
По умолчанию удалит ресурс в Корзину. Чтобы удалить ресурс не помещая в корзину, следует указать параметр permanently=true. Если удаление происходит асинхронно, то вернёт ответ со статусом 202 и ссылкой на асинхронную операцию. Иначе вернёт ответ со статусом 204 и пустым телом.

Параметры

(required)
Путь к файлу или папке.

Список возвращаемых атрибутов.

query	string

Выполнить асинхронно.

query	boolean
md5	
md5 удаляемого файла.

query	string
permanently	
Удалить ресурс не помещая в Корзину.
