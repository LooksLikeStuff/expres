Особенности реализованной системы логирования:

  • Глобальное логирование: Все действия со сделками (создание, редактирование, удаление) записываются в единую таблицу логов
  • Детальная информация: Каждый лог содержит IP адрес, User Agent, тип действия, описание и JSON с изменениями
  • Расширенные фильтры: Поиск по пользователю, типу действия, дате, ID сделки и содержимому изменений
  • Только для администратора: Доступ к глобальным логам имеют только пользователи со статусом "admin"
  • Автоматическое логирование: Система Observer автоматически создает логи при любых изменениях в моделях Deal
  • Сохранение удаленных данных: При удалении сделки вся информация сохраняется в логе для возможности восстановления
