/* === ПРИМЕЧАНИЯ === */

1. Все форматы поддерживают загрузку файлов до 1.5 ГБ
2. Рекомендуется использовать архивы для больших проектов
3. Некоторые форматы могут требовать специального ПО для просмотра
4. PDF рекомендуется для финальных версий документов
5. Для 3D моделей предпочтительны универсальные форматы (OBJ, FBX, STEP)
